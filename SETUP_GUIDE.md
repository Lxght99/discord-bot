# Setup Guide for Discord Bot

## Adding the Bot to Your Server
1. **Visit the Bot's URL**: Go to the URL provided by the bot creator to invite the bot to your server.
2. **Select Server**: From the drop-down menu, select the server you have the right to add the bot to.
3. **Authorize Permissions**: Check the permissions that the bot will require and authorize them. Ensure to grant necessary permissions for full functionality.
4. **Complete Authorization**: Click the **Authorize** button and complete any CAPTCHA challenge that may appear.

## Using the Control Panel
1. **Access the Control Panel**: Navigate to the web address of the control panel that is linked in the bot’s documentation.
2. **Login**: Enter your credentials to log in. If this is your first time, you may need to create an account.
3. **Configure Settings**: Once logged in, you can configure settings as per your requirements. 
   - Modify channels for alerts.
   - Set up roles for access to different features.
4. **Save Changes**: After configuring the settings, make sure to save the changes. 
5. **Test the Bot**: Go back to your Discord server and test the bot’s functionality to ensure everything is set up correctly.

## Troubleshooting
- If the bot does not appear in your server, ensure you have the correct permissions to add bots.
- Check your control panel settings if the bot is not responding as expected.
